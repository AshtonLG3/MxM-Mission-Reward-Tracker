# MxM Mission Reward Tracker

A lightweight Tampermonkey/Greasemonkey userscript that automatically:

- Detects the number of **completed missions** in Musixmatch Studio  
- Retrieves the **reward rate (USD)**  
- Calculates your **total daily earnings**  
- Converts the amount to **South African Rand (ZAR)**  
- Displays a **draggable earnings widget**  
- Updates automatically every 2 seconds
